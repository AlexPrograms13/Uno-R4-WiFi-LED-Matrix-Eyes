# Uno-R4-WiFi-LED-Matrix-Eyes
Do you want to test your Arduino Uno R4 WiFi in a fun way? Then try this script. In this script your 'LED Matrix' comes to life once it has generated the code!


1. If you want to run the code make sure you installed '**Arduino Uno R4 Boards**'
1. you can download it in the boards manager from **Arduino IDE**
